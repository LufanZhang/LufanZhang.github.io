We establish a website called igenus stock prediction which will suggest stocks for the users to either buy or sell. If clients wants to buy or sell their stocks, they can use our app to get recommendation.

We use an open source called LAMP-(linux,apache,mysql and php) to implement it.

manual selection

selection: page where the user can enter a stock symbol, investment horizon and get detailed prediction using our advanced hybrid algorithm.

Genius prediction

page where the user just needs to choose a desired risk tolerance level and investment time horizon. Our iGenius backend will smartly analyze all the stocks and pick the ones with highest gains within the chosen risk tolerance. We call google chart to show the candlestick chart on each page.

Convenient alert

salert feature that allows users to set an alert on stocks of their interest. Our iGenius stock screener automatically analyzes the chosen stocks every day and sends alert email to the user when the analysis shows there is a BUY signal for given stock.
