greedy snake AI

this project is written by python, I import curese module, it provide us a clean user interface, it is used to show the process of how the greedy snake AI move in the board.The complexity of this project is algorithm. How to let snake eat more food with safe path until the body of snake fill the board.

Because not only we need to find the shortest path that snake can eat the food, but also we need to consider the safety of the snake, we created a virtual snake to help real snake test the safety of the path.

Use the BFS to find if there has path between snake's head and the food, if the answer is yes, still cannot move real snake directly, because each move the snake makes, the situation of the board will be changed, which means the situation will be changed from safe to dangerous, so use the virtual snake to eat the food, and check if there is a safe path between virtual snake's head and snake's tail.

If there is a path between snake's head and the food, but the virtual snake tell us that it is not the safe way, treat is as that there is no path between head and food.

Determin if there is a path between snake's head and snake's tail, if the answer is yes, move to the tail use the longest way, if he answer is no, just use the wandering method.
